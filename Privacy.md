## ina Privacy Center

Welcome to **ina** ("we", "us", or "our").

Your privacy is important to us. This Privacy Policy explains what information we collect, why we collect it, and how we protect it while providing a reliable Discord music experience.

We only collect the minimum amount of data necessary for the bot to function.

## 1. Information We Collect

### 1.1 User Data

| Data Type | Purpose | Details |
|-----------|---------|---------|
| Discord User ID | Account identification & premium linking | Unique Discord identifier |
| Listening History | Recently played tracks | Optional feature |
| Favorites | Store saved songs | Only tracks you explicitly save |
| Spotify Profile | Spotify integration | Optional connection |
| Playlists | Custom and collaborative playlists | Names, tracks, collaborators |

---

### 1.2 Server (Guild) Data

| Data Type | Purpose |
|-----------|---------|
| Guild ID | Save server-specific settings |
| Configuration | Prefix, volume, 24/7 mode, ignored channels |
| Queues | Maintain playback state between sessions |

## 2. How We Use Your Information

The information collected is used strictly for bot functionality.

### Core Features
- Play music in voice channels
- Manage queues and playlists
- Maintain consistent playback

### Personalization
- Remember saved tracks
- Provide better music experience
- Support optional integrations

### Premium Services
- Verify subscriptions
- Enable premium features
- Manage subscription benefits
